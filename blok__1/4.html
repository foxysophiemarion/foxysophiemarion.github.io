Настройка сборки «Чертогов Фрилансера» 4

ЧФ4 построена так, чтобы максимально большое количество возможностей можно было настроить под потребности пользователя.

Для настройки сборки открываем файл template.config.js в папке проекта.

Список настроек:

1. vscode — Настройка редактора VS Code

1.1 settings — Автоматическая настройка псевдонимов для проекта (плагин Path Autocomplete)

1.2 snippets — Автоматическое добавление сниппетов ЧФ4 в проект

2. navpanel — Настройка навигационной панели

2.1 dev — В режиме разработчика. Включатель: true/false

2.2 build — В режиме продакшена. Включатель: true/false

2.3 position — Позиция вывода панели: left/right

2.4 color — Цвет текста и ссылок

2.5 background — Цвет фона

2.6 transition — Скорость анимации открытия в миллисекундах

3. server — Настройка локального сервера

3.1 path — Настройка путей к файлам: ./ — относительный, / — абсолютный

3.2 copyfiles — Копирование файлов из папки src/files в dist/files в режиме продакшена. Включатель: true/false

3.3 version — Добавление уникальной версии к конечным CSS/JS файлам. Включатель: true/false

3.4 hostname — Название сервера

3.5 port — Порт сервера

4. html — Настройка обработки HTML-файлов

4.1 beautify — Форматирование в режиме продакшена

4.2 enable — Включатель: true/false

4.3 indent — Настройка типа отступов

5. styles — Настройка работы со стилями и обработки SCSS/CSS-файлов

5.1 tailwindcss — Подключить Tailwindcss. Включатель: true/false

5.2 pxtorem — Переводить PX в REM в режиме продакшена. Включатель: true/false

5.3 critical — Создать и добавить в HTML-файлы критические CSS-стили. Включатель: true/false

5.4 codesplit — Разделять CSS-файлы на отдельные и подключать соответствующим HTML-страницам в режиме продакшена. Включатель: true/false

5.5 devfiles — Создание несжатых копий файлов в режиме продакшена. Включатель: true/false

6. fonts — Настройка обработки шрифтов

6.1 iconsfont — Создание шрифта из SVG-иконок (иконного шрифта) и подключение его в проект. Включатель: true/false

6.2 download — Загружать и делать локальными шрифты, которые подключены удаленно (например, из Google Fonts). Включатель: true/false

7. images — Настройка обработки изображений

7.1 svgsprite — Создание спрайта из SVG-иконок и подключение его в проект. Включатель: true/false

7.2 optimize — Настройка оптимизации изображений в режиме продакшена

7.3 enable — Включатель: true/false. Если оптимизация отключена, изображения просто копируются без конвертации и оптимизации

7.4 edithtml — Вносить изменения в HTML-файлы. Если включено, HTML-файлы будут изменены в соответствии с настройками конвертации и создания дополнительных размеров изображений

7.5 sizes — Размеры файлов, которые будут дополнительно созданы

7.6 dpi — Множители для создания дополнительных файлов

7.7 attrignore — Атрибут тега IMG, при наличии которого изображение не будет обработано

7.8 modernformat — Настройка конвертации в современные форматы

7.9 enable — Включатель: true/false

7.10 type — Формат изображения, в который нужно конвертировать: webp/avif

7.11 only — Оставлять только файлы в современном формате после конвертации. Включатель: true/false

7.12 quality — Качество изображения

7.13 jpeg — Настройка оптимизации изображений

7.14 quality — Качество изображения

7.15 png — Настройка оптимизации изображений

7.16 quality — Качество изображения

8. js — Настройка обработки JavaScript-файлов

8.1 hotmodules — «Горячее» подключение компонентов. Включатель: true/false

8.2 devfiles — Создание несжатых копий файлов в режиме продакшена. Включатель: true/false

8.3 bundle — Настройка формирования конечного файла в режиме продакшена

8.4 enable — Система собирает весь код в один JS и один CSS файл независимо от настройки styles -> codesplit. Включатель: true/false

9. react — Включает обработку React. Включатель: true/false

10. vue — Включает обработку Vue. Включатель: true/false

11. php — Настройка PHP-сервера

11.1 enable — Включатель: true/false

11.2 base — Путь к папке, где находятся файлы PHP

11.3 hostname — Имя хоста

11.4 port — Порт

11.5 binary — Путь к папке, где установлен интерпретатор PHP

11.6 ini — Путь к файлу настроек php.ini

12. pug — Настройка препроцессора PUG

12.1 enable — Включатель: true/false

13. ftp — Настройка отправки конечных файлов (из папки dist) на сервер (хостинг) по FTP в режиме продакшена

13.1 host — Имя FTP хоста

13.2 port — FTP порт

13.3 remoteDir — Удаленная папка на сервере

13.4 user — Имя пользователя FTP

13.5 password — Пароль пользователя FTP

14. logger — Логи работы модулей и компонентов сборки

14.1 terminal — Логи работы модулей в терминале. Включатель: true/false

14.2 console — Логи работы компонентов в консоли

14.3 enable — Включатель: true/false

14.4 removeonbuild — Удаляет логи в режиме продакшена. Включатель: true/false

15. projectpage — Создание страницы проекта в режиме продакшена. Страница будет содержать ссылки на все HTML-файлы проекта для удобной демонстрации заказчику

15.1 enable — Включатель: true/false

15.2 projectname — Название проекта, которое выводится на странице вместо %projectname%. Если пусто, заменяется названием папки проекта

15.3 template — Путь к шаблону страницы проекта

15.4 outfilename — Имя файла. Если пусто, то создается HTML-файл с названием папки проекта

16. aliases — Настройка псевдонимов

17. coffee — Настройка сообщения о перерыве

17.1 enable — Включатель: true/false

17.2 text — Текст сообщения

17.3 interval — Интервал вывода сообщения в минутах
